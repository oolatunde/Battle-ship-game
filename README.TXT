PROJECT TITLE: Battle Ship
PURPOSE OF PROJECT: Attacks on ships between 2 players. Ships are located in different positions
on the boards. Assumption: First to hit 3 ships wins the game
If a player hits, the board updates with letter "H"
if a player misses, the board updates with a sign "+"
VERSION or DATE:September 15th
HOW TO START THIS PROJECT: Run the BattleShip program in any JVM
AUTHORS: John Opeyemi
USER INSTRUCTIONS: Different ships are denoted with separate characters. Hit points do not defer on the type of ship.
Any hit on a ship is one(1) point.
